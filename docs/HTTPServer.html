    HttpServer

    HTTPServer is a custom implementation of a server to handle the HTTP protocol. It implements native TCP sockets through Node's Net class.

    Constructor

    Parameters:

    port Number <optional>
    		 8124 Local port where the server will be run from.

    Type:
    Object
    A new HTTPServer instance.
    Example
    // Initializing a new server instance.
const server = new HTTPServer();
server.init();
